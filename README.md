# Port-Scanner
Fast Multi-threaded Port Scanner
Command line tool designed for service discovery and basic network auditing. This project is part of my development in the area of ​​cybersecurity and the domain of Python applied to networks.

Main features:
- Speed: Threading implementation (ThreadPoolExecutor) for fast parallel scans.
- Flexibility: Allows you to define port ranges and number of threads through arguments.
- DNS Resolution: Supports both IP addresses and domain names.
- Clean code: Use of context managers (with) for efficient management of network resources.
